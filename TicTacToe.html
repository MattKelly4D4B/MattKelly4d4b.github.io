Impossible Tic-Tac-Toe!

This game of Tic-Tac-Toe is controlled by an UNBEATABLE AI!

The script running this game uses the minimax algorithm to correctly decide which space will give a favorable outcome. It essentally scores all the possible moves it can make on a positive scale and the opponents moves on a negative scale. It is called minimax because the algorithm seeks to maximize the minimum gain from each move. Starting from the last possible move in the game, the algorithm backtracks the moves and will determine which moves to choose next.
In game theory this equates to winning almost every time.
